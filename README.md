# Piezoelectric Microphone

## Overview
A functional microphone built using piezoelectric sensors that convert sound 
vibrations into electrical signals, exploring the relationship between 
sound frequency, vibration amplitude, and sensor response.

## Objectives
- Build a working microphone using piezoelectric sensing elements
- Investigate sensor response to varying sound frequencies and amplitudes
- Apply principles of waves, vibrations, and material behavior

## Tools & Technologies
- LTSpice
- Circuit Design
- Electronics & Sensing

## Key Learnings
- Piezoelectric effect and its practical applications
- Signal conversion from mechanical to electrical domain
- Circuit analysis and simulation using LTSpice
